# v0.1.0 - Agent 工作台可演示版

> 这是 GitHub Release 文案草案，不是最终 tag。  
> 正式发布前，请确认代码已经合并到 `main`，并完成最终验收。

## Summary

`v0.1.0` 是当前仓库首个以“Agent 工作台可演示版”为口径的正式 release 草案。

这一版的核心不是简单换壳，而是把新的前端工作台宿主、agent 对话主线、testcase 工作区和平台基础管理页收敛成一条可展示、可回归、可继续开发的里程碑版本。

## Highlights

- `apps/platform-web-vue` 已作为新的前端工作台宿主落地
- `chat / sql-agent / testcase / threads` 主线已具备可演示能力
- 平台基础管理页已完成首轮迁移并接入真实平台接口
- 顶栏系统区、列表页母版、分页、下拉、弹层、dark mode 与响应式细节已统一收敛
- 演示账号、烟测清单、汇报口径和遗留项已形成正式文档

## Added

- 新增 `platform-web-vue` 前端宿主与正式工作区布局
- 新增 `overview / projects / users / assistants / me / security / audit` 首批平台页面
- 新增 `chat` 通用对话工作台与 `sql-agent` 统一基座
- 新增 `testcase/generate / cases / documents` 三个子页迁移
- 新增 `threads` 点选后再加载详情的懒加载模式
- 新增 `Resources / Playbook` 作为前端开发范式与模板沉淀入口
- 新增发版、演示与验收相关文档

## Changed

- 将前端正式宿主从旧迁移试验态收敛到 `apps/platform-web-vue`
- 将顶栏系统区统一为项目切换、语言切换、公告中心、用户菜单四类交互
- 将列表页母版统一为 `DataTable / Pagination / 列设置 / 筛选设置 / ActionMenu / BulkActionsBar`
- 将对话工作台统一到同一套执行态视图、消息级操作与上下文抽屉模型
- 对 dark mode、移动端与大屏布局做了专项复查与修正

## Fixed

- 修复线程页一次性展开所有 thread 内容导致的卡顿问题
- 修复 testcase 文档页在线预览与原始下载主链路
- 修复 chat 中 retry/edit 分支切换与默认最新分支挂载点问题
- 修复多处顶栏、下拉、分页与筛选器交互不一致问题
- 修复关键页面在手机和大屏下的横向溢出与未占满问题
