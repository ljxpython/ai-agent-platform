# Runtime Web 文档导航

文档类型：`Docs Index`

`apps/runtime-web` 是当前仓库里的 **runtime 调试前端**。  
它不是正式平台宿主，也不是控制面治理入口。

推荐读路径：

1. `standards/runtime-web-debug-standard.md`
2. `../README.md`
3. 仓库级 `docs/development-paradigm.md`

如果你想判断：

- 这个需求是不是应该放在 `runtime-web`
- `runtime-web` 是否应该承接某个交互或验证行为
- 它和 `platform-web-vue`、`runtime-service` 的边界怎么分
